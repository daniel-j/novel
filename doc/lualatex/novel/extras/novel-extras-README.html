README for novel extras

This folder contains items that may be helpful, but are not essential to operation of the novel document class.

File novel-example.tex is a fake short novel. If you wish to compile it, place a copy in your user home folder (or other place where you have read/write privilege) and compile it there. On some operating systems, you cannot compile the original in its extras folder.

File novel-testsuite.tex exercises the various capabilities of the document class. You can look at the code, and see how it compiles. You may also change settings, to see what they do. It requires file novel-testimage.png. If you wish to compile it, place a copy (along with image) in your user home folder (or other place where you have read/write privilege) and compile it there. On some operating systems, you cannot compile the original in its extras folder.

File novel-cover-template.tex may be used for creating a cover, once you have a CMYK image. If you wish to compile it, place a copy in your user home folder (or other place where you have read/write privilege) and compile it there. On some operating systems, you cannot compile the original in its extras folder.

File novel-decoglyphs.pdf shows the decorative characters in the NovelDeco.otf file, along with the codes needed to retrieve them.

File novel-Gallery-ChapterStart.pdf shows a number of different ways to begin a chapter, making use of the ChapterStart environment and related commands.

File CPmodified.otf is not (normally) for use in TeX. I use it when editing my work in a word processor, prior to TeX markup. To use it, install the font in your operating system (not in TeX) where a word processor can find it. This font is nearly-monospaced, with exaggerated punctuation marks. I have found that punctuation errors are most frequently overlooked during editing, since they are so small in many fonts, and the spell-check software doesn't find the errors.

Included icc color profiles:

Note that the ink limit file, mentioned in novel documentation, cannot be included here, as the license does not allow bundling with other software (such as the novel document class). You have to go get it yourself. Links provided in the documentation.

File srgb.icc is a re-named copy of file sRGB-elle-V2-srgbtrc.icc by Elle Stone.

File crgb.icc is a re-named copy of file ClayRGB-elle-V2-srgbtrc.icc by Elle Stone.

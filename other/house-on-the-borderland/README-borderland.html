The House on the Borderland - demonstration files

Text Block

The file "borderland.tex" and files in its accompanying folder "borderland-components" are a demonstration of the novel document class for LuaLaTeX.

The files use UNIX-type line endings, so if you are viewing them on Windows, you need a compatible text editor (not Windows Notepad).

File "borderland.pdf" is the compiled result, as PDF/X-1a:2001 and US Web Coated SWOP v2 profile (not embedded). This is with the dimensions and layout of the as-distributed files, resulting in 124 pages, 5.5"x8.5".

Note that this text is not the complete novel. Portions have been omitted, and others edited. Thus, it is not an authoritative edition.

Cover Artwork

File "borderland-cover.png" is dimensioned to fit the text block for a Perfect-Print softcover, at the as-distributed dimensions and layout.

According to one widely-used American P.O.D. service, 0.0025" spine width is required per book page, using creme paper. With 124 pages (62 leaves) and 0.0025" per page (0.005" per leaf), the spine must be 0.31" wide. The same service requires bleed of 0.125" at all four edges.

The resulting cover image must be 11.56" wide, 8.75" high. At 300pdi, it is 3468 x 2625 pixels.

The RGB png image can be converted to CMYK jpg, using SWOP color profile at 240% ink limit, according to the instructions in the novel class documentation. You will need the srgb.icc and 240.icc color profiles. Then the converted image can be converted to PDF/X-1a:2001 with the US Web Coated SWOP v2 profile (not embedded), using borderland-cover.tex. The result is file borderland-cover.pdf.
